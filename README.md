## Quick Start
```
# Clone
git clone https://github.com/Yuvraj0157/simpleTodoApp.git

# Install
npm i

# Set .env file

# Start
npm start
```

## Tech Stack
Frontend: HTML, CSS, JS, Ejs

Backend: Nodejs, Expressjs

Database: MySQL
